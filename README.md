# Internal Tool - Project & Task Management

## Project Overview

A MERN stack application for managing projects and tasks with user authentication.

## Tech Stack

- **Backend**: Node.js, Express, MongoDB, JWT
- **Frontend**: React (Vite), React Router, Axios
- **Deployment**: Render (Backend), Vercel/Netlify (Frontend)

## Local Setup

### Backend Setup

```bash
cd backend
npm install
cp .env.example .env  # Add your MongoDB URI and JWT secret
npm run dev
```
